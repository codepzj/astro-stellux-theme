## 什么是索引

索引是一种mysql用来优化查询效率的一种数据结构【一般为二叉树】，存储在磁盘当中



```sql
select * from student where age=18;
```

这一行需要在数据库的第一行开始匹配进行全表扫描，所以效率比较低



然后加索引后就会在数据库中维护一个多叉树的数据结构，如:

![image-20250829114955134](https://cdn.jsdelivr.net/gh/codepzj/images@main/20250829115015738.png)

只需要少次判断即可查询到对应的数据

优点：建立索引，可以通过多叉树去快速找到数据，避免全表扫描，大大提高查询效率

缺点：在增删改等操作需要额外的去维护这个多叉树，造成性能开销
