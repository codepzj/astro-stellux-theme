## 运行项目

### CLI运行

```plain
kratos run
```

### **手动编译**

```plain
go build -o ./bin ./...
./helloworld -conf ../configs
```

## proto生成模版

### client

```plain
kratos proto add api/shortvid-service/v1/file.proto
```

### server

```plain
kratos proto server api/shortvid-service/v1/file.proto -t app/shortvid-service/internal/service
```
